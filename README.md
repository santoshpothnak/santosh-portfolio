# Santosh Pothnak | Lead Data & AI Architect Portfolio

A clean, professional, and minimalist personal portfolio website showcasing over 16 years of expertise in High-Scale Data Engineering, BI Architectures, and Generative AI Solutions. 

The website utilizes a premium, dark-mode visual aesthetic inspired by Google Gemini, featuring glowing background ambient orbs and an interactive particle constellation animation.

## 🚀 Live Demo & Hosting
The website is built entirely using vanilla web standards (HTML5, CSS3, and JavaScript) with no compile or build steps required. It can be hosted instantly on any static file hosting service, such as **Hostinger**, **GitHub Pages**, **Vercel**, or **Netlify**.

## 🛠️ Technology Stack
* **Markup:** Semantic HTML5
* **Styling:** Vanilla CSS3 (utilizing modern nesting, variables, and `@layer` scoping)
* **Scripting:** Vanilla JavaScript (ES6+)
* **Icons:** [Lucide Icons](https://lucide.dev/) (via CDN and inline SVGs)
* **Animation:** HTML5 `<canvas>` with `requestAnimationFrame` for high-performance physics particle constellation rendering
* **Contact Form:** [Web3Forms API](https://web3forms.com/) for serverless form processing direct to inbox

## ✨ Key Features
1. **Interactive Background Constellation:** Interactive particle nodes floating in the background, dynamically linking to one another and tracking pointer/cursor positions on desktop screens.
2. **Mobile First & Optimized:** Custom re-stacking grid structures and layout arrangements, including lightweight inline SVGs for drawer buttons, providing a fast 60fps scrolling experience on mobile browsers (Safari/Chrome iOS) with optimized battery usage.
3. **Interactive Experience Timeline:** A detailed vertical timeline highlighting career history across industry engagements (Persistent, Accenture, Infosys, and Seal Infotech) with custom bolding metrics.
4. **Filterable Projects Grid:** Client project cards that can be categorized and filtered dynamically (AI, BI, Data Quality, Migrations) using native JavaScript filtering.
5. **Community & Leadership:** Dedicated highlighting of your Udemy Instructor profile (60,000+ students) and community speaker sessions (Microsoft Fabric Bootcamps, IIM Raipur, ANU, CBIT).

## ⚙️ Configuration & Form Setup
To make the contact form fully operational on your own domain:
1. Obtain a free Access Key at [Web3Forms](https://web3forms.com/).
2. Open `index.html` and locate the hidden input field:
   ```html
   <input type="hidden" name="access_key" value="YOUR_ACCESS_KEY_HERE">
   ```
3. Replace `YOUR_ACCESS_KEY_HERE` with your key and save the file.
